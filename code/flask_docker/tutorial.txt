
TUTORIAL:
https://blog.logrocket.com/build-deploy-flask-app-using-docker/

WYMAGANIA:
1. Python >= 3.8
2. Docker (automatyczna instalacja ze skryptu)

1. Uruchom komendę:
        sh requirements.sh
    uruchomi się plik przeznaczony dla dockera, ale zawiera tez podstawowe pakiety jak Flask czy sam docker
2.Export ustawien flaska jest domyslnie w setup_frontend.sh, jesli wyjdzie error, to ponizej w "KOMENDY z palca"
3.Uruchom skrypt run_docker.sh. Kontener się zbuduje i uruchomi
4.Wyłączanie kontenera:
        docker container ls
        docker container stop [CONTAINER_ID]
5.Ponowne uruchomienie:
        docker container start [CONTAINER_ID]


KOMENDY z palca:
## Flask
pip3 install flask

export FLASK_APP=hello.py
export FLASK_ENV=development
flask run
